# Quasar App (example-ios)

A Quasar Project

## Install the dependencies
```bash
yarn
# or
npm install
```

### Start the app in xcode
```bash
quasar dev -m capacitor -T ios
```


### Build the app for production
```bash
quasar build -m capacitor -T ios
```

### Technology requriement
- capacitor v4
- xcode 13
- cocoapods 1.16
taget ios 13 min
